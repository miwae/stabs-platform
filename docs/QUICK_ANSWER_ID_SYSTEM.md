## 🚀 COMPLETE SYSTEM ARCHITECTURE

```
ID Document (Personalausweis, Reisepass, etc.)
         ↓
    Scanner/Kamera (in Stabs-Platform)
         ↓
    Python OCR Service (OFFLINE!)
    ├─ Image Preprocessing (OpenCV)
    ├─ Tesseract Text Extraction
    └─ Entity Recognition (Name, DOB, etc.)
         ↓
    Node.js API
    ├─ Duplicate Detection
    ├─ Data Validation
    └─ PDF Generation (ReportLab)
         ↓
    Vue.js Component
    ├─ Preview & Edit Fields
    ├─ Manual Review
    ├─ Confidence Scores
    └─ Print/Save
         ↓
    PostgreSQL (Encrypted)
    ├─ Original scans (pgcrypto)
    ├─ Extracted data
    ├─ Audit logs (DSGVO)
    └─ Backups (encrypted)
```
